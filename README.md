# Blood-Domain
#### Go to the Master branch to see the code. 
It is an android application to connect donor and consumer to find blood based on cosumers location. The application is named Blood_Domain.apk. This project was done as a group project in Software Engineering course. There are three more collaborators. Here are some snippets of the app.
#### Tools: Android Studio, SQLite
